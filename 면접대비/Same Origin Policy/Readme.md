# Same Origin Policy

: 동일 출처 정책

> SOP는 한 Origin에서 로드된 문서 또는 스크립트가 다른 Origin의 리소스와 상호 작용할 수 있는 방법을 제한하는 중요한 보안 메커니즘이다.
> 보안을 위협하는 문서를 격리하여, 보안 위협으로부터 보호할 수 있다.
>
> \*한마디로 말해서 웹 브라우저에서 동작하는 프로그램은 로딩된 위치에 있는 리소스만 접근 할 수 있다는 정책이다.
