# MSW(Mock Service Worker)

### Mock Service Worker란?

MSW 는 서브스 워커를 사용하여 네트워크 호출을 가로채는 API 모킹(mocking) 라이브러리이다. 쉽게 말해, 브라우저에 기생해서 마치 백엔트 API 인 척하면서 프론트 요청에 따라 가짜 데이터를 응답해주는 녀석!
